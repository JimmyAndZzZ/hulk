# hulk
### 主要特性
- 基于Mysql协议，可以通过navicat、jdbc等连接方式
- 支持用SQL查询非结构化和结构化数据库，例如Mysql、Oracle、Clickhouse、Excel、elasticsearch
- 支持不同数据源之间表关联查，目前支持inner join和left join
- 基于calcite进行SQL解析，并对SQL进行扩展，支持自定义扩展操作
- 基于aviator表达式进行自定义函数扩展
- 支持分库分表、权限设置(可到字段级别)
